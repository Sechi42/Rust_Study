# Notas sobre Cargo

- **Crear un proyecto**: Usa `cargo new <nombre_del_proyecto>`.
- **Compilar el proyecto**: Ejecuta `cargo build`.
- **Crear y ejecutar el proyecto**: Usa `cargo run`.
- **Checar errores sin generar un binario**: Ejecuta `cargo check`.
- **Directorio de resultados del build**: Los resultados se guardan en `target/debug/` en lugar del directorio del proyecto.
- **Compilación optimizada** para producción: Usa `cargo build --release`, que guarda los resultados en `target/release/`.
